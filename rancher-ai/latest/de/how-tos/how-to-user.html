Interaktion mit Liz (KI-Assistentin)

Erste Schritte mit Liz

Liz enthält einen persistierenden KI-Assistenten namens Liz. Sie können den Assistenten jederzeit öffnen, indem Sie auf das Liz-Symbol in der Benutzeroberfläche klicken.

Wenn Sie das seitliche Fenster öffnen, begrüßt Sie Liz mit Vorschlägen basierend auf dem Kontext Ihrer aktuellen Seite. Während eines Gesprächs bietet Liz:

  • Zwei vorgeschlagene Aktionen zur Fortsetzung des aktuellen Workflows.

  • Eine erkundende Aktion um mehr über verwandte Themen zu erfahren.

Context-Awareness

Liz versteht genau, was Sie sich ansehen. Jede Ressource mit einem Statusabzeichen verfügt über eine Frag Liz Schaltfläche.

  1. Klicken Sie auf das Frag Liz-Schiebeschaltfläche-Symbol neben einer Ressource.

  2. Das seitliche Fenster wird mit einem maßgeschneiderten Prompt geöffnet, der automatisch für diese spezifische Ressource generiert wurde.

  3. Liz sammelt automatisch den Ressourcennamen, die Cluster-ID und den Namespace, um genaue Hilfe zu bieten.

Aktionen bestätigen

Sicherheit hat Priorität.

Wenn Sie Liz oder einen ihrer spezialisierten Agenten bitten, Ihre Umgebung zu ändern (z. B. eine Implementierung zu aktualisieren oder eine Clusterkonfiguration zu ändern), wird Liz ein Bestätigungsdialogfeld anzeigen.

Sie müssen auf Bestätigen klicken, bevor Änderungen an Ihrem System vorgenommen werden.

Verwalten von Gesprächen

Sie können mehrere Threads verwalten oder Ihren Verlauf für später speichern.

  • Neuen Chat erstellen: Um eine neue Sitzung zu starten, klicken Sie auf das Neuer Chat-Symbol.

  • Verlauf herunterladen: Um einen Verlauf Ihres Gesprächs zu speichern, klicken Sie auf das Herunterladen Symbol. Dies speichert eine .txt Datei des Transkripts auf Ihrem lokalen Laufwerk.

Agenten wechseln

Standardmäßig verwendet Liz "Intelligente Weiterleitung", um Ihre Anfrage an den besten spezialisierten Agenten zu senden (z. B. einen Sicherheitsagenten für Schwachstellen).

Wenn Sie manuell mit einem bestimmten Agenten sprechen möchten:

  1. Klicken Sie auf das Agentenauswahl Symbol.

  2. Wählen Sie den gewünschten Agenten aus der Liste aus.

Wenn Liz bemerkt, dass Sie häufig denselben Agenten verwenden, kann sie vorschlagen, für diese Sitzung in den manuellen Modus zu wechseln.
