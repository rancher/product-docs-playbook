Kurzanleitung

Diese Kurzanleitung erklärt, wie man Ollama verwendet, um LLM-Funktionen für Liz bereitzustellen. Sie können nach der ursprünglichen Installation die Anbieter wechseln, indem Sie sich auf Anleitungen für Administratoren beziehen.

Voraussetzungen

Die aktuelle Version des KI-Assistenten erfordert die folgenden Komponenten:

  • Helm installiert

  • Rancher Prime/Community Manager 2.13 oder höher

Stellen Sie sicher, dass Sie die Rechte haben, CRD (cluster_admin) im Host-Cluster bereitzustellen.

Installieren Sie Liz: The Rancher AI Assistant

Die Installation von Liz ist ein zweistufiger Prozess:

  • Setzen Sie den Agenten und das MCP über das bereitgestellte Helm-Chart ein.

  • Setzen Sie die Rancher UI-Erweiterung ein.

Installieren Sie den Agenten und das MCP im lokalen Cluster

  1. Erstellen Sie eine values.yaml-Dateieinstellung (für Ollama):

    ollamaLlmModel: "gpt-oss:20b"
ollamaUrl: "http://ollama:11434"
activeLlm: "ollama"

    Sie können diese Einstellungen und Anbieter später in den „globalen Einstellungen → KI-Assistent-Tabs“ von Rancher ändern.

  2. Installieren Sie das rancher-ai-agent Chart

    helm install rancher-ai-agent \
  --namespace cattle-ai-agent-system \
  --create-namespace \
  -f values.yaml oci://registry.suse.com/rancher/charts/rancher-ai-agent

Installieren Sie die UI-Erweiterung

Öffnen Sie die Rancher Manager UI und navigieren Sie zur Seite 'Erweiterungen'.

  1. Klicken Sie im Rancher Manager auf Erweiterung in der Menüleiste.

  2. Verwenden Sie das Drei-Punkte-Menü in der oberen rechten Ecke und wählen Sie 'Rancher-Repositories hinzufügen' aus.

  3. Wählen Sie 'Offizielles Rancher-Erweiterungs-Repository hinzufügen', um das Repository hinzuzufügen.

  4. Klicken Sie auf 'Hinzufügen'.

  5. Suchen Sie die Karte des KI-Assistenten und klicken Sie auf 'Installieren'.

  6. Wählen Sie eine Version aus (oder verwenden Sie standardmäßig die neueste) und klicken Sie auf 'Installieren'.

  7. Sobald die Erweiterung installiert ist, klicken Sie auf die Schaltfläche 'Neu laden', die oben auf der Seite erscheint.
