Ce démarrage rapide explique comment utiliser Ollama pour fournir des capacités LLM à Liz. Vous pouvez changer de fournisseur après l’installation initiale en vous référant à Procédures pour l’administrateur.

Conditions préalables

La version actuelle de l’Assistant IA nécessite les composants suivants:

  • Helm installé

  • Rancher Prime/Community Manager 2.13 ou supérieur

Assurez-vous d’avoir les droits pour déployer CRD (cluster_admin) sur le cluster hôte.

Installez Liz: The Rancher AI Assistant

L’installation de Liz est un processus en deux étapes:

  • Déployez l’agent et le MCP via le chart Helm fourni.

  • Déployez l’extension Rancher UI.

Installez l’Agent et le MCP sur le cluster local

  1. Créez un fichier de configuration values.yaml (pour Ollama) :

    ollamaLlmModel: "gpt-oss:20b"
ollamaUrl: "http://ollama:11434"
activeLlm: "ollama"

    Vous pouvez modifier ces paramètres et fournisseurs plus tard dans les onglets « Paramètres globaux → Assistant IA » de Rancher.

  2. Installez le chart rancher-ai-agent

    helm install rancher-ai-agent \
  --namespace cattle-ai-agent-system \
  --create-namespace \
  -f values.yaml oci://registry.suse.com/rancher/charts/rancher-ai-agent

Installez l’extension Rancher UI

Ouvrez l’interface utilisateur de Rancher Manager et accédez à la page 'Extensions'.

  1. Dans Rancher Manager, cliquez sur Extension dans la barre de menu.

  2. Utilisez le menu à trois points en haut à droite et sélectionnez 'Ajouter des dépôts Rancher'.

  3. Sélectionnez 'Ajouter le dépôt officiel des extensions Rancher' pour ajouter le dépôt.

  4. Cliquez sur 'Ajouter'.

  5. Trouvez la carte de l’Assistant IA et cliquez sur 'Installer'.

  6. Sélectionnez une version (ou utilisez la dernière par défaut) et cliquez sur 'Installer'.

  7. Une fois l’extension installée, cliquez sur le bouton 'Recharger' qui apparaît en haut de la page.
