快速入门

此快速入门说明如何使用 Ollama 为 Liz 提供 LLM 功能。您可以在初始安装后通过参考 管理员操作指南 切换提供者。

先决条件

当前版本的 AI 助手需要以下组件：

  • 已安装 Helm

  • Rancher Prime/Community Manager 2.13 或更高版本

确保您在主集群上具有部署 CRD（cluster_admin）的权限。

技术要求

以下是支持的 AI 组件及其要求的表格。

您只需满足您打算运行的特定组件的要求。要求将根据您选择部署的特定大语言模型（LLM）而有所不同。

如果您运行自己的 Ollama，请确保至少有 gpt-oss:20b 用于本地模型。请参考 Ollama 文档 了解如何拉取模型。

安装 Liz: The Rancher AI Assistant

安装 Liz 是一个两步过程：

  • 通过提供的 Helm 图表部署代理和 MCP。

  • 部署 Rancher UI 扩展。

在本地群集上安装代理和 MCP

  1. 创建 values.yaml 设置文件（用于 Ollama）：

    ollamaLlmModel: "gpt-oss:20b"
ollamaUrl: "http://ollama:11434"
activeLlm: "ollama"

    您可以稍后在 Rancher 的“全局设置 → AI 助手选项卡”中更改这些设置和提供者。

  2. 安装 rancher-ai-agent 图表

    helm install rancher-ai-agent \
  --namespace cattle-ai-agent-system \
  --create-namespace \
  -f values.yaml oci://registry.suse.com/rancher/charts/rancher-ai-agent

安装 UI 扩展

打开 Rancher 管理 UI 并导航到“扩展”页面。

  1. 在 Rancher 管理器上，点击菜单栏中的扩展。

  2. 使用右上角的三点菜单，选择“添加 Rancher 储存库”。

  3. 选择 '添加官方 Rancher 扩展储存库' 以添加该储存库。

  4. 单击 '添加'。

  5. 找到 AI 助手卡片并单击 '安装'。

  6. 选择一个版本（或默认使用最新版本）并单击 '安装'。

  7. 扩展安装完成后，单击页面顶部出现的 '重新加载' 按钮。
