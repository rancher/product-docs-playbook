Erstellen von persistentem Speicher in Amazon EBS

Dieser Abschnitt beschreibt, wie man Amazons Elastic Block Store in EC2 einrichtet.

  1. Wechseln Sie in der EC2-Konsole zum Abschnitt ELASTIC BLOCK STORE im linken Bereich und klicken Sie auf Volumes.

  2. Klicken Sie auf Volume erstellen.

  3. Optional: Konfigurieren Sie die Größe des Volumes oder andere Optionen. Das Volume sollte in derselben Verfügbarkeitszone erstellt werden wie die Instanz, an die es angeschlossen wird.

  4. Klicken Sie auf Volume erstellen.

  5. Wählen Sie Schließen.

Ergebnis: Persistenter Speicher wurde erstellt.

Weitere Informationen zur Einrichtung des neu erstellten Speichers in Rancher finden Sie im Abschnitt Einrichtung des bestehenden Speichers.
