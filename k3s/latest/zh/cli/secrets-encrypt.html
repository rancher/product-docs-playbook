  1. дҪҝз”Ёж Үеҝ— --secrets-encryption еҗҜеҠЁ K3s жңҚеҠЎеҷЁ

    еҪ“еүҚ _дёҚ_ж”ҜжҢҒеңЁжІЎжңүеҠ еҜҶзҡ„жғ…еҶөдёӢеҗҜеҠЁ K3s е№¶еңЁзЁҚеҗҺеҗҜз”ЁгҖӮ

    $ k3s secrets-encrypt status
Encryption Status: Enabled
Current Rotation Stage: reencrypt_finished

  2. еңЁS1дёҠиҪ®жҚўеҜҶж–ҮеҠ еҜҶеҜҶй’Ҙ

  3. зӯүеҫ…йҮҚж–°еҠ еҜҶе®ҢжҲҗгҖӮжҹҘзңӢжңҚеҠЎеҷЁж—Ҙеҝ—пјҢжҲ–зӯүеҫ…пјҡ

    $ k3s secrets-encrypt status
Encryption Status: Enabled
Current Rotation Stage: reencrypt_finished

    K3sе°ҶжҜҸз§’йҮҚж–°еҠ еҜҶзәҰ5дёӘеҜҶж–ҮгҖӮе…·жңүеӨ§йҮҸеҜҶж–Үзҡ„йӣҶзҫӨеҸҜиғҪйңҖиҰҒеҮ еҲҶй’ҹжүҚиғҪйҮҚж–°еҠ еҜҶгҖӮжӮЁеҸҜд»ҘеңЁжңҚеҠЎеҷЁж—Ҙеҝ—дёӯи·ҹиёӘиҝӣеәҰгҖӮ

# If using openrc

еңЁеҗҜеҠЁеёҰжңү`--secrets-encryption`ж Үеҝ—зҡ„й«ҳеҸҜз”ЁжҖ§йӣҶзҫӨеҗҺпјҢеҸҜд»ҘзҰҒз”ЁеҜҶж–ҮеҠ еҜҶгҖӮ

иҷҪз„¶дёҚжҳҜеҝ…йңҖзҡ„пјҢдҪҶе»әи®®жӮЁйҖүжӢ©дёҖдёӘжңҚеҠЎеҷЁиҠӮзӮ№жқҘиҝҗиЎҢ`secrets-encrypt`е‘Ҫд»ӨгҖӮ

дёәз®ҖжҙҒиө·и§ҒпјҢжң¬жҢҮеҚ—дёӯдҪҝз”Ёзҡ„дёүеҸ°жңҚеҠЎеҷЁе°Ҷиў«з§°дёәS1гҖҒS2гҖҒS3гҖӮ

иҰҒеңЁй«ҳеҸҜз”ЁжҖ§йӣҶзҫӨдёҠзҰҒз”ЁеҜҶж–ҮеҠ еҜҶпјҡ

  1. еңЁS1дёҠзҰҒз”Ё

    k3s secrets-encrypt disable

  2. дҪҝз”ЁзӣёеҗҢзҡ„еҸӮж•°жқҖжӯ»е№¶йҮҚеҗҜS1гҖӮеҰӮжһңе°ҶK3sдҪңдёәжңҚеҠЎиҝҗиЎҢпјҡ

    # If using systemd
systemctl restart k3s
# If using openrc
rc-service k3s restart

  3. дёҖж—ҰS1еҗҜеҠЁпјҢжқҖжӯ»е№¶йҮҚеҗҜS2е’ҢS3гҖӮ

  4. еңЁS1дёҠдҪҝз”Ёж Үеҝ—йҮҚж–°еҠ еҜҶ

    k3s secrets-encrypt reencrypt --force --skip

иҰҒеңЁй«ҳеҸҜз”ЁжҖ§йӣҶзҫӨдёҠйҮҚж–°еҗҜз”ЁеҜҶж–ҮеҠ еҜҶпјҡ

  1. еңЁ S1 дёҠеҗҜз”Ё

    k3s secrets-encrypt enable

  2. дҪҝз”ЁзӣёеҗҢзҡ„еҸӮж•°жқҖжӯ»е№¶йҮҚеҗҜS1гҖӮ

  3. дёҖж—ҰS1еҗҜеҠЁпјҢжқҖжӯ»е№¶йҮҚеҗҜS2е’ҢS3гҖӮ

  4. еңЁS1дёҠдҪҝз”Ёж Үеҝ—йҮҚж–°еҠ еҜҶ

    k3s secrets-encrypt reencrypt --force --skip

еҜҶж–ҮеҠ еҜҶзҠ¶жҖҒ

еҜҶж–ҮеҠ еҜҶе·Ҙе…·еҢ…еҗ«дёҖдёӘ status е‘Ҫд»ӨпјҢз”ЁдәҺжҳҫзӨәиҠӮзӮ№дёҠеҪ“еүҚеҜҶж–ҮеҠ еҜҶзҡ„зҠ¶жҖҒдҝЎжҒҜгҖӮ

еҚ•жңҚеҠЎеҷЁиҠӮзӮ№дёҠзҡ„е‘Ҫд»ӨзӨәдҫӢпјҡ

$ k3s secrets-encrypt status
Encryption Status: Enabled
Current Rotation Stage: start
Server Encryption Hashes: All hashes match

Active  Key Type  Name
------  --------  ----
 *      AES-CBC   aescbckey

й«ҳеҸҜз”ЁжҖ§йӣҶзҫӨдёӯзҡ„еҸҰдёҖдёӘзӨәдҫӢпјҢеңЁж—ӢиҪ¬еҜҶй’ҘеҗҺпјҢдҪҶеңЁйҮҚеҗҜжңҚеҠЎеҷЁд№ӢеүҚпјҡ

$ k3s secrets-encrypt status
Encryption Status: Enabled
Current Rotation Stage: rotate
Server Encryption Hashes: hash does not match between node-1 and node-2

Active  Key Type  Name
------  --------  ----
 *      AES-CBC   aescbckey-2021-12-10T22:54:38Z
        AES-CBC   aescbckey

жҜҸдёӘйғЁеҲҶзҡ„иҜҰз»ҶдҝЎжҒҜеҰӮдёӢпјҡ

  • еҠ еҜҶзҠ¶жҖҒпјҡжҳҫзӨәиҠӮзӮ№дёҠжҳҜеҗҰзҰҒз”ЁжҲ–еҗҜз”Ё Secrets еҠ еҜҶ

  • еҪ“еүҚж—ӢиҪ¬йҳ¶ж®өпјҡжҢҮзӨәиҠӮзӮ№дёҠзҡ„еҪ“еүҚж—ӢиҪ¬йҳ¶ж®өгҖӮ
    йҳ¶ж®өдёәпјҡstartгҖҒprepareгҖҒrotateгҖҒreencrypt_requestгҖҒreencrypt_activeгҖҒreencrypt_finished

  • жңҚеҠЎеҷЁеҠ еҜҶе“ҲеёҢпјҡеҜ№дәҺй«ҳеҸҜз”ЁжҖ§йӣҶзҫӨпјҢиҝҷжҢҮзӨәжүҖжңүжңҚеҠЎеҷЁзҡ„жң¬ең°ж–Үд»¶жҳҜеҗҰеӨ„дәҺзӣёеҗҢйҳ¶ж®өгҖӮиҝҷеҸҜд»Ҙз”ЁжқҘзЎ®е®ҡеңЁиҝӣе…ҘдёӢдёҖдёӘйҳ¶ж®өд№ӢеүҚжҳҜеҗҰйңҖиҰҒйҮҚеҗҜжңҚеҠЎеҷЁгҖӮеңЁдёҠйқўзҡ„й«ҳеҸҜз”ЁжҖ§зӨәдҫӢдёӯпјҢиҠӮзӮ№ 1 е’ҢиҠӮзӮ№ 2 зҡ„е“ҲеёҢдёҚеҗҢпјҢиЎЁжҳҺе®ғд»¬еҪ“еүҚжІЎжңүзӣёеҗҢзҡ„еҠ еҜҶй…ҚзҪ®гҖӮйҮҚеҗҜжңҚеҠЎеҷЁе°ҶеҗҢжӯҘе®ғд»¬зҡ„й…ҚзҪ®гҖӮ

  • еҜҶй’ҘиЎЁпјҡжҖ»з»“иҠӮзӮ№дёҠжүҫеҲ°зҡ„з§ҳеҜҶеҠ еҜҶеҜҶй’Ҙзҡ„дҝЎжҒҜгҖӮ

    • жҙ»еҠЁпјҡ"*" иЎЁзӨәеҪ“еүҚз”ЁдәҺеҜҶж–ҮеҠ еҜҶзҡ„еҜҶй’ҘпјҲеҰӮжһңжңүзҡ„иҜқпјүгҖӮKubernetes дҪҝз”Ёжҙ»еҠЁеҜҶй’ҘжқҘеҠ еҜҶд»»дҪ•ж–°еҜҶж–ҮгҖӮ

    • еҜҶй’Ҙзұ»еһӢпјҡдҪҝз”ЁжӯӨе·Ҙе…·зҡ„жүҖжңүеҜҶй’ҘйғҪжҳҜ AES-CBC зұ»еһӢгҖӮжҹҘзңӢжӣҙеӨҡдҝЎжҒҜ иҝҷйҮҢгҖӮ

    • еҗҚз§°:еҠ еҜҶеҜҶй’Ҙзҡ„еҗҚз§°гҖӮ

etcd-snapshot д»ӨзүҢ
