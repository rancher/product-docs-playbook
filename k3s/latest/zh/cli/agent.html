代理的节点标签和污点

K3s 代理可以使用选项 --node-label 和 --node-taint 进行配置，这将为 kubelet 添加标签和污点。这两个选项仅在注册时添加标签和/或污点，因此只能添加一次，之后无法通过运行 K3s 命令进行更改。

以下是一个示例，展示如何添加标签和污点：

     --node-label foo=bar \
     --node-label hello=world \
     --node-taint key1=value1:NoExecute

如果您想在节点注册后更改节点标签和污点，您应该使用 kubectl。有关如何添加 污点 和 节点标签 的详细信息，请参阅官方 Kubernetes 文档。
