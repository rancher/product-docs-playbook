SUSE® Rancher Prime: OS Manager オペレーター Helm チャート

SUSE® Rancher Prime: OS Manager Operator は、SUSE® Rancher Prime: OS Manager バージョンの管理と、エッジまたはベアメタルインストールを支援するためのマシンインベントリの維持を担当しています。

関連するチャートは、https://rancher.com/docs/rancher/v2.6/[Rancher Manager v2.6] クラスター上で Helm パッケージマネージャーを使用してエレメンタルオペレーターのデプロイメントを起動します。

前提条件

  • Rancher Manager バージョン v2.6

  • Helm クライアントバージョン v3.8.0+

Helm チャート情報を取得

helm pull oci://registry.suse.com/rancher/elemental-operator-chart
helm show all oci://registry.suse.com/rancher/elemental-operator-chart

チャートをインストール

helm install --create-namespace -n cattle-elemental-system elemental-operator-crds \
             oci://registry.suse.com/rancher/elemental-operator-crds-chart
helm install --create-namespace -n cattle-elemental-system elemental-operator \
             oci://registry.suse.com/rancher/elemental-operator-chart

このコマンドは、デフォルト設定で Kubernetes クラスターにエレメンタルオペレーターをデプロイします。

以下の 設定 を参照してください。

コマンドのドキュメントについては、https://helm.sh/docs/helm/helm_install/[helm install] を参照してください。

チャートをアンインストール

helm uninstall -n cattle-elemental-system elemental-operator

これにより、チャートに関連するすべての Kubernetes コンポーネントが削除され、リリースが削除されます。

コマンドのドキュメントについては、https://helm.sh/docs/helm/helm_uninstall/[helm uninstall] を参照してください。

チャートをアップグレードする

helm upgrade -n cattle-elemental-system \
             --install elemental-operator \
             oci://registry.suse.com/rancher/elemental-operator-chart

コマンドのドキュメントについては、https://helm.sh/docs/helm/helm_upgrade/[helm upgrade] を参照してください。

設定

インストール前のチャートのカスタマイズ を参照してください。詳細なコメント付きのすべての設定可能なオプションを確認するには、チャートの values を訪問するか、次の設定コマンドを実行してください：

`console showLineNumbers
helm show values oci://registry.suse.com/rancher/elemental-operator-chart
`
