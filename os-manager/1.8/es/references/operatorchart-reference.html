SUSE® Rancher Prime: OS Manager Gráfico de Helm del operador

El SUSE® Rancher Prime: OS Manager Operator es responsable de gestionar las versiones de SUSE® Rancher Prime: OS Manager y mantener un inventario de máquinas para ayudar con las instalaciones en el edge o en equipo sin sistema operativo.

El gráfico asociado inicia una ampliación de elemental-operator en el clúster Rancher Manager v2.6 utilizando el gestor de paquetes Helm.

Requisitos previos

  • Versión de Rancher Manager v2.6

  • Versión del cliente Helm v3.8.0+

Obtener información del gráfico de Helm

helm pull oci://registry.suse.com/rancher/elemental-operator-chart
helm show all oci://registry.suse.com/rancher/elemental-operator-chart

Instalar gráfico de Helm

helm install --create-namespace -n cattle-elemental-system elemental-operator-crds \
             oci://registry.suse.com/rancher/elemental-operator-crds-chart
helm install --create-namespace -n cattle-elemental-system elemental-operator \
             oci://registry.suse.com/rancher/elemental-operator-chart

El comando despliega elemental-operator en el clúster de Kubernetes con la configuración predeterminada.

Ver configuración a continuación.

Ver helm install para la documentación del comando.

Desinstalar gráfico de Helm

helm uninstall -n cattle-elemental-system elemental-operator

Esto elimina todos los componentes de Kubernetes asociados con el gráfico y elimina la release.

Ver helm uninstall para la documentación del comando.

Actualizando gráfico de Helm

helm upgrade -n cattle-elemental-system \
             --install elemental-operator \
             oci://registry.suse.com/rancher/elemental-operator-chart

Ver helm upgrade para la documentación del comando.

Configuración

Ver Personalizando el gráfico de Helm antes de instalar. Para ver todas las opciones configurables con comentarios detallados, visita los valores del gráfico de Helm, o ejecuta estos comandos de configuración:

`console showLineNumbers helm show values oci://registry.suse.com/rancher/elemental-operator-chart `
