Soporte de Gatekeeper

El soporte de Gatekeeper comienza a partir de estas versiones:

  • kwctl: v0.2.0

  • policy-server: v0.2.0

Gatekeeper es un proyecto dirigido a Kubernetes con características listas para usar para la integración.

Compatibilidad con directivas existentes

Todas las directivas existentes de Gatekeeper deberían ser compatibles con Admission Controller como se explica en este capítulo.

Si encuentras que esto no es cierto para tus directivas de Gatekeeper, infórmalo, y trabajaremos para asegurar que tu directiva de Gatekeeper funcione con Admission Controller.

Las directivas deben compilarse con la CLI de opa para el destino wasm.

Para la ejecución de directivas, puedes leer más sobre el soporte integrado en Admission Controller.
