Was ist SUSE® Rancher Prime Cluster API?

Die Cluster-API-Erweiterung für Rancher, auch bekannt als SUSE® Rancher Prime Cluster API, ist ein Kubernetes-Operator, der Integration zwischen Rancher Manager und Cluster API (CAPI) bietet, mit dem Ziel, vollständige CAPI-Unterstützung für Rancher zu bringen.

Mit SUSE® Rancher Prime Cluster API können Sie folgende Aufgaben ausführen:

  • Automatisches Importieren von CAPI-Clustern in Rancher, durch die Installation des Rancher Cluster Agent in CAPI-provisionierten Clustern.

  • Implementieren Sie einen GitOps-nativen Workflow zur Verwaltung von Kubernetes-Clustern in Rancher.

  • Vereinfachte Verwaltung von CAPI-Anbietern mit dem CAPI-Operator.
