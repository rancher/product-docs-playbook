Overview

What is Rancher Turtles?

The Cluster API Extension for Rancher, also known as SUSE® Rancher Prime Cluster API, is a Kubernetes Operator that provides integration between Rancher Manager and Cluster API (CAPI) with the aim of bringing full CAPI support to Rancher.

With SUSE® Rancher Prime Cluster API, you can:

  • Automatically import CAPI clusters into Rancher, by installing the Rancher Cluster Agent in CAPI provisioned clusters.

  • Implement a GitOps-native workflow to managing Kubernetes clusters in Rancher.

  • Simplified management of CAPI Providers using the CAPI Operator.

The following video shows you how to get started with SUSE® Rancher Prime Cluster API in just 3 minutes. You will learn how to:

  • Use Rancher Dashboard to install Turtles.

  • Deploy the CAPI Infrastructure Provider for Docker.

  • Provision a workload cluster via Fleet: you can inspect source files in Turtles repository.

  • Import the workload cluster into Rancher.
