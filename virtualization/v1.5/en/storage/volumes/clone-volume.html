How to Clone a Volume

After creating a volume, you can clone the volume by following the steps below:

  1. Click the ⋮ button and select the Clone option.

  2. Select clone volume data.

  3. Configure the Name of the new volume and click Create.

  4. (Optional) A cloned volume can be added to a VM using Add Existing Volume.
