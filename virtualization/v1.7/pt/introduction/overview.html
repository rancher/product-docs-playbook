Visão Geral

SUSE® Virtualization é uma solução moderna, aberta, interoperável e hiperconvergente (HCI) construída sobre o Kubernetes. É uma alternativa de código aberto projetada para operadores que buscam uma solução HCI nativa de nuvem. SUSE Virtualization opera em servidores bare metal e fornece virtualização integrada e capacidades de armazenamento distribuído. Além das máquinas virtuais (VMs) tradicionais, SUSE Virtualization oferece suporte automático a ambientes conteinerizados por meio da integração com SUSE® Rancher Prime. Ele oferece uma solução que unifica a infraestrutura virtualizada antiga e, ao mesmo tempo, permite a adoção de contêineres de locais que vão do núcleo à borda.

Arquitetura

A arquitetura consiste em tecnologias de código aberto de ponta:

  • Sistema Operacional Linux. SUSE® Rancher Prime: OS Manager para SUSE Linux Enterprise Micro está no núcleo de SUSE Virtualization e é uma distribuição Linux imutável projetada para remover o máximo possível de manutenção do SO em um cluster Kubernetes.

  • Construído sobre o Kubernetes. Kubernetes se tornou a linguagem de infraestrutura predominante em todos os fatores de forma, e SUSE Virtualization é uma solução HCI com Kubernetes por trás.

  • Gerenciamento de virtualização com KubeVirt. KubeVirt fornece gerenciamento de virtualização usando KVM sobre o Kubernetes.

  • Gerenciamento de armazenamento com SUSE Storage. SUSE Storage fornece armazenamento em blocos distribuído e gerenciamento de níveis.

  • Observabilidade com Grafana e Prometheus. Grafana e Prometheus fornecem monitoramento robusto e registro robusto.

Recursos

SUSE Virtualization é uma plataforma de infraestrutura pronta para empresas e fácil de usar, que aproveita o armazenamento local, diretamente conectado, em vez de SANs externas complexas. Utiliza a API do Kubernetes como uma linguagem de automação unificada para cargas de trabalho de contêineres e VMs. Alguns recursos principais incluem:

  • Fácil de começar. Como SUSE Virtualization é fornecido como uma imagem de appliance inicializável, você pode instalá-lo diretamente em um servidor bare metal com a imagem ISO ou instalá-lo automaticamente usando scripts iPXE.

  • Gerenciamento do ciclo de vida da VM. Crie, edite, clone e exclua VMs facilmente, incluindo injeção de chave SSH, cloud-init e console gráfico e de porta serial.

  • Migração ao vivo da VM. Mova uma VM para um host ou nó diferente sem tempo de inatividade.

  • Backup, instantâneo e restauração da VM. Faça backup de suas VMs a partir de servidores NFS, S3 ou dispositivos NAS. Use seu backup para restaurar uma VM que falhou ou criar uma nova VM em um cluster diferente.

  • Gerenciamento de armazenamento. SUSE Virtualization suporta armazenamento em blocos distribuído e gerenciamento de níveis. Volumes representam armazenamento; você pode facilmente criar, editar, clonar ou exportar um volume.

  • Gerenciamento de rede. Suporta o uso de um IP virtual (VIP) e múltiplas interfaces de rede (NICs). Se suas VMs precisarem se conectar à rede externa, crie uma VLAN ou uma rede sem marcação.

  • Integração com SUSE Rancher Prime. Acesse SUSE Virtualization diretamente pela página de Gerenciamento de Virtualização do SUSE Rancher Prime e gerencie suas cargas de trabalho de VM junto com seus clusters Kubernetes.

Interface do usuário

SUSE Virtualization fornece uma interface poderosa e fácil de usar baseada na web para visualizar e gerenciar sua infraestrutura. Uma vez instalado, você pode acessar o endereço IP da interface do usuário a partir do terminal do nó.
