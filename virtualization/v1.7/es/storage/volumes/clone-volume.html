Clonar un volumen

Instrucciones para clonar un volumen

Después de crear un volumen, puedes clonar el volumen siguiendo los pasos a continuación:

  1. Haz clic en el botón ⋮ y selecciona la opción Clone.

  2. Seleccione clone volume data.

  3. Configura el Name del nuevo volumen y haz clic en Create.

  4. (Opcional) Un volumen clonado puede añadirse a una máquina virtual utilizando Add Existing Volume.

Reconstrucción de réplicas para volúmenes clonados SUSE Storage desconectados

Cuando clonas un volumen desconectado SUSE Storage en un clúster de múltiples nodos, SUSE Storage crea solo una réplica en lugar del número esperado (por ejemplo, tres réplicas en un clúster de tres nodos). El estado del clon de volumen mostrado es copy-completed-awaiting-healthy en lugar de complete.

SUSE Storage reconstruye automáticamente las réplicas restantes después de que el volumen clonado se adjunta a una máquina virtual. Sin embargo, puedes iniciar el proceso de reconstrucción de réplicas inmediatamente para el volumen desconectado realizando cualquiera de las siguientes acciones:

  • En la interfaz de usuario de SUSE Storage, adjunta manualmente el volumen y espera a que se complete el proceso de reconstrucción de réplicas.

  • Habilita la configuración SUSE Storage offline-replica-rebuilding para permitir la reconstrucción automática de réplicas para volúmenes desconectados.

Este comportamiento se aplica exclusivamente a volúmenes desconectados. SUSE Storage realiza automáticamente la reconstrucción de réplicas para volúmenes que están adjuntos a una máquina virtual en funcionamiento.
