Descripción general

SUSE® Virtualization es una solución de infraestructura hiperconvergente (HCI) moderna, abierta e interoperable construida sobre Kubernetes. Es una alternativa de código abierto diseñada para operadores que buscan una solución HCI nativa de nube. SUSE Virtualization se ejecuta en servidores sin sistema operativo y proporciona capacidades de virtualización integrada y almacenamiento distribuido. Además de las máquinas virtuales tradicionales (VMs), SUSE Virtualization admite entornos en contenedores automáticamente gracias a la integración con SUSE® Rancher Prime. Ofrece una solución que unifica la infraestructura virtualizada heredada al tiempo que permite la adopción de contenedores tanto desde ubicaciones centrales como periféricas.

Arquitectura

La arquitectura consiste en tecnologías de código abierto de vanguardia:

  • Linux OS. SUSE® Rancher Prime: OS Manager para SUSE Linux Enterprise Micro está en el núcleo de SUSE Virtualization y es una distribución de Linux inmutable diseñada para eliminar tanto mantenimiento del sistema operativo como sea posible en un clúster de Kubernetes.

  • Construido sobre Kubernetes. Kubernetes se ha convertido en el lenguaje de infraestructura predominante en todos los factores de forma, y SUSE Virtualization es una solución HCI con Kubernetes bajo el capó.

  • Gestión de virtualización con KubeVirt. KubeVirt proporciona gestión de virtualización utilizando KVM sobre Kubernetes.

  • Gestión de almacenamiento con SUSE Storage. SUSE Storage proporciona almacenamiento en bloques distribuido y jerarquización.

  • Observabilidad con Grafana y Prometheus. Grafana y Prometheus proporcionan monitoreo y registro robustos.

Funciones

SUSE Virtualization es una plataforma de infraestructura lista para empresas y fácil de usar que aprovecha el almacenamiento local, conectado directamente, en lugar de complejos SAN externos. Utiliza la API de Kubernetes como un lenguaje de automatización unificado para cargas de trabajo de contenedores y máquinas virtuales. Algunas características clave incluyen:

  • Fácil de empezar. Dado que SUSE Virtualization se envía como una imagen de appliance arrancable, puedes instalarlo directamente en un servidor sin sistema operativo con la imagen ISO o instalarlo automáticamente utilizando scripts de iPXE.

  • Gestión del ciclo de vida de máquinas virtuales. Crea, edita, clona y elimina VMs fácilmente, incluyendo inyección de clave SSH, cloud-init y consola de puerto gráfico y serial.

  • Migración en vivo de VMs. Mueve una VM a un host o nodo diferente sin tiempo de inactividad.

  • Copia de seguridad, instantánea y restauración de VMs. Realiza copias de seguridad de tus VMs desde servidores NFS, S3 o dispositivos NAS. Utiliza tu copia de seguridad para restaurar una máquina virtual fallida o crear una nueva máquina virtual en un clúster diferente.

  • La gestión del almacenamiento. SUSE Virtualization soporta almacenamiento en bloques distribuido y jerarquización. Los volúmenes representan almacenamiento; puedes crear, editar, clonar o exportar un volumen fácilmente.

  • La gestión de redes. Soporta el uso de una IP virtual (VIP) y múltiples tarjetas NIC (NICs). Si tus máquinas virtuales necesitan conectarse a la red externa, crea una VLAN o una red sin etiquetar.

  • Integración con SUSE Rancher Prime. Accede a SUSE Virtualization directamente a través de la página de Gestión de Virtualización de SUSE Rancher Prime y gestiona tus cargas de trabajo de máquinas virtuales junto a tus clústeres de Kubernetes.

Interfaz de usuario

SUSE Virtualization proporciona una interfaz web potente y fácil de usar para visualizar y gestionar tu infraestructura. Una vez instalado, puedes acceder a la dirección IP de la interfaz de usuario desde la terminal del nodo.
