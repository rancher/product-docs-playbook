Unterstützte Units für Diagramme

Diese Seite enthält alle von Diagrammen unterstützten Units. Für jede Unit wird zunächst die Beschreibung der Unit genannt, gefolgt vom Namen der Unit, die in den SUSE Observability APIs verwendet werden sollte. Um beispielsweise einen Wert in wissenschaftlicher Notation anzuzeigen, verwenden Sie sci als Wert für die Unit in einer Metrikbindung.

Die Darstellung der Unit passt sich automatisch an die Größe des Wertes an. Wenn die Unit für ein Diagramm beispielsweise bytes ist, wird der Wert als bytes(IEC) interpretiert. Ein Wert von 100 wird als 100B dargestellt, aber ein Wert von 5242880 (5*1024*1024 wird als 5MiB angezeigt.
