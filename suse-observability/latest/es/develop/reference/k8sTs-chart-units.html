Unidades soportadas para gráficos

Esta página contiene todas las unidades soportadas por los gráficos. Para cada unidad, primero se menciona la descripción de la unidad, seguida del nombre de la unidad que debe utilizarse en las APIs de SUSE Observability. Por ejemplo, para mostrar un valor en "notación científica", utiliza sci como el valor para la unidad en un enlace métrico.

La representación de la unidad se ajusta automáticamente al tamaño del valor. Por ejemplo, si la unidad para un gráfico es bytes, el valor se interpreta como bytes(IEC). Un valor de 100 se representará como 100B, pero un valor de 5242880 (5*1024*1024 se mostrará como 5MiB.
