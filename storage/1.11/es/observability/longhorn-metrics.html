Métricas de SUSE® Storage para la monitorización

Volumen

Nombre Descripción Ejemplo

longhorn_volume_actual_size_bytes

Espacio actual utilizado por cada réplica del volumen en el nodo correspondiente

longhorn_volume_actual_size_bytes{pvc_namespace="default",node="worker-2",pvc="testvol",volume="testvol"} 1.1917312e+08

longhorn_volume_capacity_bytes

Tamaño configurado en bytes para este volumen

longhorn_volume_capacity_bytes{pvc_namespace="default",node="worker-2",pvc="testvol",volume="testvol"} 6.442450944e+09

longhorn_volume_state

Estado del volumen. Esta métrica utiliza la etiqueta state para indicar el estado actual del volumen. El valor es 1 para el estado actual y 0 para otros. Estados: creando, adjunto, separado, adjuntando, separando, eliminando

longhorn_volume_state{pvc_namespace="default",node="worker-2",pvc="testvol",volume="testvol",state="attached"} 1

longhorn_volume_robustness

Robustez del volumen. Esta métrica utiliza la etiqueta state para indicar la robustez actual. El valor es 1 para el estado actual y 0 para otros. Estados: desconocido, saludable, degradado, con fallos

longhorn_volume_robustness{pvc_namespace="default",node="worker-2",pvc="testvol",volume="testvol",state="healthy"} 1

longhorn_volume_read_throughput

longhorn_volume_write_throughput

Rendimiento de escritura de este volumen (Bytes/s)

IOPS de lectura de este volumen

longhorn_volume_write_iops

IOPS de escritura de este volumen

Esta métrica indica que el volumen está ahora en modo de solo lectura. La métrica es 1 o ningún registro para cada volumen

CSI

El componente sidecar de CSI tiene métricas integradas para que los usuarios obtengan información sobre las operaciones de CSI. Las métricas de operaciones de CSI cubren el conteo total, el conteo de errores y la latencia de llamadas. Longhorn habilita las métricas añadiendo la bandera --http-endpoint para cada componente sidecar de CSI. Puedes usar PodMonitor de Prometheus para recopilar estas métricas.

Nombre Puerto

longhorn-csi-attacher

8000

longhorn-csi-provisioner

8000

longhorn-csi-resizer

8000

longhorn-csi-snapshotter

8000
