Estado del volumen. Esta métrica utiliza la etiqueta state para indicar el estado actual del volumen. El valor es 1 para el estado actual y 0 para otros. Estados: creando, adjunto, separado, adjuntando, separando, eliminando

CSI

El componente sidecar de CSI tiene métricas integradas para que los usuarios obtengan información sobre las operaciones de CSI. Las métricas de operaciones de CSI cubren el conteo total, el conteo de errores y la latencia de llamadas. Longhorn habilita las métricas añadiendo la bandera --http-endpoint para cada componente sidecar de CSI. Puedes usar PodMonitor de Prometheus para recopilar estas métricas.

Nombre Puerto

longhorn-csi-attacher

8000

longhorn-csi-provisioner

8000

longhorn-csi-resizer

8000

longhorn-csi-snapshotter

8000
